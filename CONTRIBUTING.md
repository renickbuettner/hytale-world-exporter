## How to Contribute

### Bug Reports
If you find a bug, please open an issue with:
- A clear description of the problem
- Steps to reproduce the issue
- Your operating system (Windows/macOS)

### Feature Ideas
Have an idea for a new feature? Open an issue and describe:
- What the feature should do
- Why it would be useful

### Pull Requests
Bug fixes and new features are welcome! Please:
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push to the branch (`git push origin feature/my-feature`)
5. Open a Pull Request

## Development Setup

```bash
# Clone the repository
git clone https://github.com/renickbuettner/hytale-world-exporter.git
cd hytale-world-exporter

# Build the project
cargo build

# Run the application
cargo run
```

## Code Style
- Follow Rust conventions
- Run `cargo fmt` before committing
- Ensure `cargo clippy` passes without warnings
